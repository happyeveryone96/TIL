<h1>비교</h1>
<h2>연산자</h2>
<li>값에 대해서 어떤 작업을 컴퓨터에게 지시하기 위한 기호</li>
<pre>
대입 연산자(이항 연산자)
=   
비교 연산자
<li>==(동등 연산자(equal operator)), >> true,false >> boolean       ===(일치 연산자(strict equal operator)</li>
<li> >, <, <=, >=  </li>
</pre>
<br>
<pre>
alert(null == undefined);       //true
alert(null === undefined);      //false
alert(true == 1);               //true
alert(true === 1);              //false
alert(true == '1');             //true
alert(true === '1');            //false
 
alert(0 === -0);                //true
alert(NaN === NaN);             //false
</pre>
