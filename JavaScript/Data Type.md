<h1>Data Type(자료형)</h1>
<h2>String & Number</h2>
<div>Number(숫자)는 따옴표 없이 작성, String(문자)은 따옴표 안에 작성</div>
<div>str.toUpperCase();   >>  대문자로 변경</div>
<div>str.indexOf();   >>  문자 위치 찾기</div>
<div>str.trim();   >>  공백 삭제</div>
